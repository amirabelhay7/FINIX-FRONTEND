Frequently asked questions

Quick answers to common questions about FINIX and our modules.

What is FINIX?
FINIX is a modular finance platform that brings together credit, scoring, wallet, insurance, repayment and vehicle collateral in one ecosystem. It is designed for lenders and insurers who need a single, scalable solution.
Which modules can I use?
You can start with one or more of: Credit, Scoring, Wallet, Insurance, Repayment and Vehicles. Each module can be enabled independently and integrated via APIs.
How do I get access to the backoffice?
Admin access to the backoffice dashboard is granted to users with an admin role. Contact your administrator or sign in with an account that has been assigned admin privileges.
Is there an API?
Yes. All modules expose REST APIs for integration with your existing systems. Documentation and API keys are available after onboarding.
